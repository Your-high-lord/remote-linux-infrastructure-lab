# 🚀 Remote Linux Infrastructure Lab

## Overview

This project demonstrates a complete remote Linux infrastructure lab built using Kali Linux. The lab was created to practice Linux administration, networking, remote access, virtualization, and web server deployment.

---

## Technologies Used

- Kali Linux
- VirtualBox
- Apache2
- SSH
- Tailscale
- NoMachine
- HTML
- Git & GitHub

---

## Features

- Remote SSH access from Android using Termux
- Secure private networking using Tailscale
- Remote desktop access using NoMachine
- Apache web server hosting a custom webpage
- Linux system administration
- Git version control

---
